可能是最全的玩客云刷机教程&我遇到的坑

提示：刷机有风险，操作需谨慎。

前段时间闲来无事刷到了玩客云的视频，就去闲鱼买了个。价格42（记得那个视频里面很多人30在出，我买的时候涨到40多了，现在60不包邮了…这还不如买N1和我家云了）。

2023-5-27：现在看起来很便宜，但是考虑到玩客云的Soc…还是建议上别的机子

实测裸机待机功耗在1.5w左右，建议40以下入手。

准备

需要的工具 作用
镊子 拿来短接
USB_Burning_Tool 刷入Firmware
USB-Write/USB image Tool 刷写Armbian到U盘的工具
Armbian 玩客云的系统
U盘 None
PuTTY 使用ssh链接玩客云
一个铁片/撬棒 拆机
USB公对公数据线 传输Firmware

Start

把铁片捅到sd卡槽的缝隙里，就可以取下I/O装饰板，拧下螺丝，就可以取下主板（主板有四个铁钩，固定主板的，捅一下即可，这个铁钩断了也没事，反正不影响使用）

然后看SD卡槽处有无V1.3的丝印，有的话就是新版本的板子，没有的话就是老版本的板子

然后再安装USB_Burning_Tool，然后把USB公对公的数据线一头插在靠近HDMI接口的USB接口，然后再把另一头插在电脑的USB接口上，找到菜单，文件-导入烧录包-选择s805_flash_snail.img。

拿出你的镊子，短接主板的短接点（具体看下图，第一张图为老版短接点，第二张图为新版短接点），然后再接上电源，这里注意插电源的时候也要短接，镊子不能松！！！！！

（建议在短接之前把电源给玩客云插上，然后把插头断开，短接的时候一只手短接，一只手去插插头，如果你有那种有开关的排插的话，那就先把玩客云的插头接上去，不开开关，到时候再按按钮打开排插）

然后玩客云就会进入刷机模式，USB_Burning_Tool会自动刷入Firmware，以后刷机就不需要拆开短接刷机了。

这里提一下，如果USB_Burning_Tool卡住刷机失败，可以尝试：

  • 快速插拔USB公对公线

  • 更换USB接口

  • 台式机的话可以试试电脑后置的usb接口

  • 可以把线插到USB2.0的接口

  • 换台电脑

    ![RB9ZQ7`AWF4_B4_4J6UYE_J.png](http://ivanstar.gitee.io/photo/wky-sj/jPIW7cXnKwVtAqs.png)

    刷完后如上图所示，如果你尝试了各种方法还是无法刷机的话，那就是的玩客云的eMMC可能有坏块，建议刷入精简版底包试试（链接放在文末）

然后，将你的U盘接上电脑，打开Usb Image Tool

选中你的U盘，点击Restore，

选择镜像，软件就会自动将镜像写入U盘。

当进度条到达100％后，关掉软件，拔下U盘，把U盘插到玩客云上，插网线，插电源，过一会玩客云的IP就会出现在路由器的管理页面

下面是水星路由器，玩客云的IP会直接显示

如果你的路由器是梅林固件，点客户端状态就能看到

访问这个IP

你就可以看到玩客云导航，然后我们就可以把玩客云的系统刷入内置的eMMC了。

打开Putty，在Host Name那里填入玩客云的ip，我们就能通过SSH连接玩客云

填写完IP后点击Open，后面会弹出一个对话框，选择是

账号：root

密码：powersee233

输入./install.sh即可把系统刷入玩客云内置的eMMC，刷入完成后会掉线重启，此时拔下U盘即可。
